Welcome! Here you can test Oltartkʰica font by inputting arbitrary Ithkuil using the input method.

In order to input an Ithkuil character, type its code and select the desired variant using space or number keys. Diacritics are shown on top of a placeholder character in the choices, but only the diacritic itself is input. It will be appended to the text in the area above. If the area has text cursor inside, click outside it to activate the input method.

Most codes correspond to abbreviation of the grammar categories listed in the Grammar. Consonantal codes correspond to their transliteration without the diacritics. Additions and exceptions are as follows:

  • FRA and UNF for patterns and stems in Framed and Unframed
  • TER for Tertiary
  • Doubled versions for ejective consonants
  • GG for glottal stop, MM for consonantal gemination, ZZ for Ż
  • DEG for Suffix degrees
  • IL for Illocution, MO for Personal Reference Adjunct and Mood, PHA for Phase, SA for Sanction
  • VER for standard and REV for alternate Version
  • XX for quotes and placeholders
